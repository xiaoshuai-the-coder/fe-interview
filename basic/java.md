# Java 相关

## 基础

- **集合**：`ArrayList/LinkedList/HashMap/ConcurrentHashMap` 的数据结构、扩容、线程安全与典型坑。
- **并发**：线程模型、锁（`synchronized`/`ReentrantLock`）、CAS、线程池参数与队列策略。
- **JVM**：内存模型、GC（新生代/老年代）、常见垃圾回收器、OOM 排查思路。

## 工程与实践

- **性能与稳定性**：慢 SQL/慢接口定位、缓存策略、限流熔断降级、日志与链路追踪。
- **线上排障**：常用命令与工具（如 jstack/jmap/jstat）、从现象到定位的排查路径。
